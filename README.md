# PulseGuide

Demo player for the [PulseMap](https://github.com/hartphoenix/pulsemap) protocol — synced lyrics, chords, and interactive playback for any mapped song.

## What is PulseGuide?

PulseGuide is a web app and Chrome extension that reads PulseMap files and synchronizes display components (lyrics, chords, sections) to audio playback from any supported platform.

## Development

bun install
bun run dev
bun run typecheck
